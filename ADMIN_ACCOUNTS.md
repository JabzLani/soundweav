## How to Access the Admin Dashboard

1. **Sign Up / Login**: Navigate to the SoundWeave platform and sign up using the admin email address
2. **Access Admin Panel**: Once logged in as an admin, navigate to `/admin` or look for the admin dashboard link
3. **Role-Based Access**: Your account will automatically have admin privileges based on your assigned role

### Admin Dashboard URL
```
https://[your-soundweave-domain]/admin
```

## Security Guidelines

1. **Keep Credentials Secure**: Do not share admin credentials via unsecured channels
2. **Change Passwords**: Encourage team members to update their passwords after first login
3. **Two-Factor Authentication**: Enable 2FA when available for added security
4. **Audit Logs**: All admin actions are logged for accountability
5. **Regular Reviews**: Periodically review admin access and permissions
